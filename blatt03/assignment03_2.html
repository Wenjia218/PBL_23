All plotting types in R:

1, Histogram:

hist(data, main = topic, xlab = xais, ylab = yaix, (col = color, border = bordercolor))

suitable for continuous data types visualize the distribution of a continuous variable

2, density plot

density(data, (main = , xlab =, col = color1))

suitable for continuous data types visualize the distribution of a variable by displaying its density

3, line plot

plot(a, b, type = l)

suitable for continuous data types visualize the trend of a continuous variable over time or some other variable

4, scatter plot

plot(a, b)

suitable for continuous data types visualize the relationship between two variables

5, box plot

boxplot(value1 ~ value2 ( vector/formula ), data, notch, varwidth, names, main)

suitable for continuous data types visualize the distribution of a continuous variable by displaying the quartiles

6, bar plot

barplot(y, names.arg = x)

suitable for categorical data types compare the values

7, heatmaps

heatmap(data, scale = none)

use custom color set: col<- colorRampPalette(c(red, white, blue))(256)

suitable for categorical data types visualize the relationship between two variables

8, pie charts:

pie(vector, labels, radius, main, col, clockwise)

suitable for categorical data types display the proportion of each category in a categorical variable

how to choose xlab, ylab, title and configure textsize, margin size and number of plots displayed:

using following commands:

xlab = lables_of_values ylab = lables_of_values main = title

text size: cex = 1.5 margin size: par(mar = c(5, 4, 4, 2) + 0.1)) display two plots side by side: par(mfrow = c(1,2))

what are the types of data:

Qualitative Data Type: Nominal Ordinal

Quantitative Data Type: Discrete Continuous

Continuous - numeric data can have any value within a range. Examples include time, height and temperature.

Discrete - numeric data that canonly have certain values. Examples include shoe size, number of people in a room and the number of marks on a test.

Categoric - the data arewords. Examples include colour such as red or blue, and how an object feels like, eg: rough or smooth.
